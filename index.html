What this detector checks

  • Duplicate fingerprints (same device voting more than once)
  • Fingerprint used with different voter IDs
  • Multiple votes from the same IP address
  • Multiple votes at the same timestamp

Paste data in CSV format with headers: fingerprint,voter_id,candidate,ip_address,timestamp.
