Fake Vote Detector UI Guide

Learn how the app interface works and how fingerprint-based vote detection is applied.

App UI Overview

The application has three main sections:

  • Input panel — paste or enter vote data in CSV format.
  • Control buttons — analyze the data or load the sample dataset.
  • Results area — view suspicious activity detected by the analyzer.

Fingerprint-based detection

This project uses a fingerprint field to identify the voting device or browser. That means the app can detect fake votes even when the voter_id field is changed.

  • Duplicate fingerprints indicate repeated voting from the same device.
  • Different voter IDs with the same fingerprint indicate possible identity fraud.
  • Multiple records from the same IP or timestamp can indicate automated voting.
